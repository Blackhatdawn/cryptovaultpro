================================================================================
                        COMMON ISSUES & QUICK FIXES
================================================================================

Issue: MongoDB not found/installed
Fix:   Install from https://www.mongodb.com/try/download/community

Issue: "ModuleNotFoundError: No module named 'fastapi'"
Fix:   cd backend && pip install -r requirements.txt

Issue: "Address already in use" on port 8001
Fix:   Change port in backend/.env or kill existing process

Issue: Backend crashes immediately
Fix:   Check .env file exists and MONGO_URL is correct

Issue: Frontend shows ECONNREFUSED errors
Fix:   Ensure backend is running (curl http://localhost:8001/ping)

Issue: Port 27017 already in use (MongoDB)
Fix:   Change MONGO_URL in .env to different port

More issues? See complete troubleshooting in:
  → PHASE_1_STARTUP_GUIDE.md (Troubleshooting section)
